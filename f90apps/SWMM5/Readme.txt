source5-0-022.zip
=================
Contains the source code for the computational engine
of SWMM 5.0.022. Consult the included Roadmap.txt file
for an overview of the various code modules.

makefiles.zip
=============
Contains "make" files for compiling the SWMM 5.0 engine
on various platforms, including Microsoft C/C++ 6, 
Microsoft C++ 2005, and the GNU C compiler for Linux.
