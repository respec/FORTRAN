HTSINSI
  • ALLOC - Allocate rows in the inpad and generate tsget/tsput instructions

  • CHAIN - Sort records in workfl

  • CHANGE - Change any subsequent appearances of a source or target to refer to a given inpad row

  • OSVFPT - Assign values to the flag-pointers for the time series in each operation in this exgrp

  • OSVPRO - Process a group of records from workfl, converting inpad row assignments to osv flag-pointer values

  • RELESE - Release all rows flagged for release

  • ROW - Find the number of the first available row in the inpad (or expad). return this value and mark the row "unavailable"

  • SAME - Chain all occurrences of the same value in specified fields in workfl

  • SOURCE - Process entries from the network or ext targets block, considering operation as a source

  • TARGET - Process entries in workfl, derived from ext sources or network block, considering operation as a target

  • TSINS - Write a primitive tsget/tsput instruction

  • WKDMP1 - Dump a specified range of the workfl to messu

  • WkDMP2 - Dump a set of primitive tsget or tsput instructions
