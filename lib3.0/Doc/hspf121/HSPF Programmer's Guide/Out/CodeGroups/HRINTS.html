HRINTS
  • HRITSI - Read in general timseries data from message file

  • GPFINT - Initialize the instruction files for tsget/tsput by placing the appropriate error offsets plus other data in the initial record of each instruction file

  • HRIMSI - Read in mass link default data from information file

  • FINSTR - Finish the tsget/tsput instruction

  • NETBLK - Expand and check any entries for a given exgrp, in the network block. chain entries by source and target opn sequence order.

  • OPNTS - Process a reference to time series pertaining to operation opno, which were specified in, or implied by, a user's instruction

  • MEMTS - Process a single member

  • IVALUE - Look at a supplied integer value. if it's positive, return the value, if negative return the value in the osv with a virtual origin equal to its absolute value.

  • PAIRS - Check pairs of entries in source and target tables and write paired entries to workfl

  • SCHBLK - Expand and check any entries for a given exgrp, in the schematic block.

  • SRCBLK - Expand and check entries in ext sources block, for a given range of operations, and chain by target opn sequence order

  • TARBLK - Expand and check any entries in the ext targets block for this exgrp. chain entries by source opn sequence order.
