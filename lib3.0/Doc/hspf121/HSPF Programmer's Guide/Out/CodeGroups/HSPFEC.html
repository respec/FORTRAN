HSPFEC
  • TIMSER - Process the blocks which deal with time series

  • GTTMRC - Get an entry from time series blocks data structure

  • PSCHEM - Expand an entry in the schematic block by using either default time series connections from the message file or user-defined connections from a mass link block

  • EXTTS - Check and expand a reference to a group of time series or a single time series to be obtained from an external source/target (trflag=1 if external target is being processed)

  • RDMFAC - Look at a supplied character value. If it is blank, return default, else return supplied real value.
