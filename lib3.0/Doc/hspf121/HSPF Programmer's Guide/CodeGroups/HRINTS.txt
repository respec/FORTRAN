<ul>
<li><a href="HRINTS/HRITSI">HRITSI</a> - Read in general timseries data from message file<p>
<li><a href="HRINTS/GPFINT">GPFINT</a> - Initialize the instruction files for tsget/tsput by placing the appropriate error offsets plus other data in the initial record of each instruction file<p>
<li><a href="HRINTS/HRIMSI">HRIMSI</a> - Read in mass link default data from information file<p>
<li><a href="HRINTS/FINSTR">FINSTR</a> - Finish the tsget/tsput instruction<p>
<li><a href="HRINTS/NETBLK">NETBLK</a> - Expand and check any entries for a given exgrp, in the network block.  chain entries by source and target opn sequence order.<p>
<li><a href="HRINTS/OPNTS">OPNTS</a> - Process a reference to time series pertaining to operation opno, which were specified in, or implied by, a user's instruction<p>
<li><a href="HRINTS/MEMTS">MEMTS</a> - Process a single member<p>
<li><a href="HRINTS/IVALUE">IVALUE</a> - Look at a supplied integer value.  if it's positive, return the value, if negative return the value in the osv with a virtual origin equal to its absolute value.<p>
<li><a href="HRINTS/PAIRS">PAIRS</a> - Check pairs of entries in source and target tables and write
paired entries to workfl<p>
<li><a href="HRINTS/SCHBLK">SCHBLK</a> - Expand and check any entries for a given exgrp, in the schematic block.<p>
<li><a href="HRINTS/SRCBLK">SRCBLK</a> - Expand and check entries in ext sources block, for a given range of operations, and chain by target opn sequence order<p>
<li><a href="HRINTS/TARBLK">TARBLK</a> - Expand and check any entries in the ext targets block for this
exgrp.  chain entries by source opn sequence order.<p>
</ul>
